This program was developed under Linux using FASM assembler.

The script `compile' makes all files required for Xlife-8-IBMPC package and puts
them to a disk image file.  It uses FASM aand several utilities (cat, sed, awk, gcc, ...).
The invocation of utility `fasm-listing' is optional and may be commented or removed.

It is required to write a name of the disk image into beginning of this script.

The file 'npi.bas' should be compiled by Microsoft Quick Basic compatible compiler.  The
result of the compilation should be renamed to 'np4.exe' and placed to the disk with
Xlife-8 distribution.

This program files should be installed to the root directory.  If you want to use it from
the arbitrary directory than use SUBST, e.g.,

cd DIRECORY-WITH-XLIFE-8
subst a: .
a:
autoexec.bat

or run attached XLIFE0.BAT script.
